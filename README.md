# Heart Disease Prediction Application
**Course:** M2 Software Development (2025-2026)  
**Group Members:** Hamza Gouaref

## Project Description
A modular data application designed to analyze heart health using a Random Forest model. 
This project follows professional Python standards including logging, type hinting, and containerization.

## How to Run
### Using Docker (Recommended)
1. Ensure Docker is installed.
2. Run: `docker-compose up --build`
3. Open `http://localhost:8501`

### Using a Virtual Environment
1. `python -m venv env`
2. `source env/bin/activate` (or `.\env\Scripts\activate` on Windows)
3. `pip install -r requirements.txt`
4. `streamlit run app/streamlit_app.py`
